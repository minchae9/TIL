## 1949. '등산로 조성'

지형의 높이를 한 군데 깎아 수정할 수 있다는 조건을 본 순간, 앞이 막막해짐을 느꼈던 문제.

그래도 이 문제를 풀면서 마냥 어려워했던 dfs 를 조금 더 잘 이해할 수 있게 되었다.

각 경우를 잘 생각해서 풀어야 한다.

dfs 에 대한 이해가 완전하지 않아 조금 장황한 풀이가 만들어졌는데 (1번 버전),

설명을 통해 구성한 2번 버전과 3번 버전을 참고하는 것이 좋다!
